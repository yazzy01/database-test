Database Queries

1. Create a database

CREATE DATABASE school_db;

2. Create tables in the database

CREATE TABLE students (
    id INT PRIMARY KEY,
    name VARCHAR(50),
    age INT,
    email VARCHAR(100)
);

CREATE TABLE courses (
    id INT PRIMARY KEY,
    name VARCHAR(50),
    hours INT
);

3. Insert data into the tables
